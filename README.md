# Breast Cancer Classification & Predictive Analysis 

## Overview
This project analyses the Breat Cancer Wisconsin dataset to classify tumours as malignant or benign using machine learning models.

## Objective
To identify key features influencing cancer diagnosis and build predictive models to support data-driven healthcare decisions.

## Dataset
Breast Cancer Wisconsin (Diagnostic) dataset  
https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data/data

## Methods
- Logistic Regression
- K-Nearest Neighbours (KNN)
- Feature selection using correlation analysis

## Results
- Logistic Regression Accuracy: 86%
- KNN Accuracy: 89%
- Key predictive feature: concave points (worst)

## Key Insights
- Concave points is a strong indicator of tumour malignancy  
- Model performance improves with feature selection  
- Predictive models can support early-stage diagnosis  

## Tools Used
- Python  
- Jupyter Notebook  
- Pandas, NumPy, Scikit-learn, Matplotlib
